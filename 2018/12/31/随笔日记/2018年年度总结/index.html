2018年年度总结

发表于 2018-12-31 | 分类于 随笔日记 | 阅读次数
undefined

2018年又踩着它的小尾巴，对着冰冷的键盘给2017年的自己一个年度汇报，给2019年的自己一个展望清单。上次看文章看到一句极其适合作为文章开头的话，结果忘记存下来了。哭唧唧，姑且用最近酷爱的番作为开篇吧（强烈推荐噢）

就算没有契机，时间也会将人的情感变为回忆。伤痛像结了疤痕一样被填满，并在没注意到的地方脱落，人就是这样不断前进的——<青春猪头少年不会梦到兔女郎学姐> 梓川咲太

2018年大概是我过去那么久的人生活得最繁杂的一年吧，大概或许是因为这是我步入社会的第一个年头。

那些工作与论文 文体开花的日子

在1月开始一直到3月份大概是我人生最为忙碌的日子了吧，甚至说是忙到晕头转向都不为过。大概承接着上一年的oss项目开始让我自己独自开发之后，整个项目的压力被压到了我身上。虽然说整体上只是属于一个内部项目，但是被关进了封闭开发的小房间里面，每天开会汇报进度，需求对接也是各位大佬直接找到我聊，然后我需要根据这样给出开发思路并且定好时间。不得不说，压力还是很大的…然而这头未完，那头又滋事。灵霸阁的项目、学校的论文全都挤在了一个时间段过来，每天的生活变成了上班写公司的代码，晚上10点下班回家赶紧打开别的项目开始写。

差点没在这段时间崩溃下来，终于灵霸阁交付上去（虽然后面政琳老哥帮忙了很多、不过说好的钱又没发，所以都不知道说是福是祸了~），然后oss项目总算告一段落，急急忙忙请假回学校拼了命撸论文。花了大概那么一周多一点完成了全部东西之后…长舒一口气，总算结束了这段多线程的工作。勉勉强强和宿友去了趟顺德作为宿舍的毕业旅游…终于感觉所有事情都结束了…

现在回想起来，那段时间真的过于疯狂了。很难想象自己到底是怎么走过来的，大概那段时候持续一两个月脑海里都只有代码吧（毕竟过年放假那几天也在写论文的代码）

语音标注以及那些语音相关的需求

结束了oss的征途，也不知是福是祸(原谅我如此用词)…我在那个准备结束封闭式开发的那天晚上被glen总点名“一起搬上13楼继续支持技术中心的事儿”。是福或许是赏识吧，那大概是对我莫大的信任才会做出这种决定了；是祸，那便是我单纯只是一个刚入门的小菜鸟，担此大任不说能否顺利完成，单单论个人能力，即便完成也是肯定有很多莫大的缺陷的吧~但是即使是这样，我在第二天还是又一次离开了我10楼的工作，跑到了13楼从此和技术中心和后台开发的大家伙混在了一起了。

语音标注平台，这是glen总当时和我画出的第一个大饼。虽然计划是对外的平台终究是没人的原因退而求次成了内部平台。虽然bug甚多，优化没有，甚至UI丑陋不堪，但是起码仍在顺利地支撑着公司大量的标注任务吧。所以即便它的确有很多不足，但是由我自己经历差不多3个多月亲手设计思考编码完成的孩子那还是将就小小嘚瑟一把吧。

后面当然就和我想法一致地演变成，我一个人对接好几个的平台需求.语音实验平台、知识分析平台…这些后面基本趋于没人维护的平台也是那段时候大家伙的想法吧，而我的一周则被每个平台瓜分出几天来实现…当然，这里其实也挺好的。因为在设计实现标注平台的时候，就会发现有一些框架层面的处理不当但是碍于项目还在不停迭代没法停下脚步。而新项目的产生则给了我实践新功能的机会吧…

当然咯，其实在语音组的工作能顺利进行少不了付老师、峰总、珊总、涂老师、周董等等各位大佬的照顾。……什么编辑距离、什么字错误率、什么是asr/vad/情感分析/角色识别、什么浏览器实现在线录音asr、什么pcm/wav、什么切片微操音频流都是在那段时间刷新着我的认知…（音频切片实时asr的确把我做到头皮发麻，查文档查到两眼发花…）

孤身一人到拥有了Pal

记得在17年的总结上，我说过“如果以后有幸和飞哥一起开发，那就真的不能太棒了。”。在大概8月份的样子吧，鉴于各种各方面的原因我被又一次踏上搬工位的旅程。回到了10楼，进入了智能管理这个开发团队（也是终于是一个有“组织”的人儿了）。

慢慢地，从一直都是独自开发，变成了合作开发；从自己茫然地开发变成了有人可以讨论请教甚至争吵；自然也从每天晚上准时下班变成了一拖再拖，一晚再晚.压力一山比一山高;从上半年的人生只有egg + vue + mysql的循环疲惫业务开发。变成了Electron/Docker/Jeknis/Graphql/Egg framework/Web Rtc/FP…一大堆的新鲜知识冲进脑海，每天都在被刷新认真的状态。

当然结果是美好的,过程却是满是苦涩的。国庆前的那些凌晨3点下班的日子以及在佛山广发欢度国庆的日子，加班加到感冒头痛，眼睛肿胀得眨眼都感觉有点刺痛…回想起来还真的蛮恐怖的，感谢活到现在的我，感谢顽强的头发以及感谢飞哥跑过来帮忙协助的开发。

Pal

2018年遇到了很多好玩的小伙伴，当然咯因为成年加班自然都是同事。被我安利得热爱起养生的付老师、成天和我讨论技术到面红耳赤还经常发我小姐姐视频的宝华哥、动不动就满嘴瓜皮的虫虫制造机、笑起来动作十分夸张喜欢说骚话的颖聪大佬、被我带坏成天说垃圾话的梦颖小姐姐、每天思考怎么给我加需求的学长伟军大佬…真的超级超级多好玩的小盆友大盆友了。（大哥这么多人我只是想稍微列举一下，没说到你别发脾气）

当然有得也有失吧，和学校里的玩伴也是动不动就几个月不联系。这次放元旦我看了看微信的聊天列表。前三个置顶的群，私聊的最新聊天记录在11月8号再往前是10月20然后是9月份的人…这意味着微信在我9月份到12月份的聊天的人不超过4个人。每次甚至都没聊啥，简直是个典型的阿宅了。完全断绝了社交。

好在勉强有和妞妞的日常、和宿舍的毕业旅行以及约饭才显得不那么寒碜

19年的展望

  1. 19年依旧希望自己能在技术这条路上越走越高越走越远吧。希望能在2020那天的总结回头看自己。“嗯，过得不丢人”
  2. 今年其实拼得有点过，身体可能已经步入亚健康了吧。下年要注意锻炼一下，迈开步子去
  3. 依旧希望可以和我家妞妞过得快乐幸福不吵架哦
  4. 希望能多找的时间和三五好友聊天说骚话，多点社交元素吧。
  5. 本年度的技术书看的有点多，来年希望能纳入看一下别的书类型。多扩展自己一下的视野吧。噢，要把今年买的藏书清理一下。
  6. 然后俗气一点赚钱钱.升职加薪哇咔咔。当然每个月划出来的学习经费一定要注意用上。同时要督促某只偷懒猪也用上。

碎碎杂谈

这次的年度总结内容有点少那是因为上一年我立了个flag。说是不再简单记录自己的事，要写成日记形式。结果洋洋洒洒写了3万字今天查阅了一下很多信息都不太适合发出来…就只能自己有起草总结了。

不能发有一些是因为为了圆自己的一些小谎言、有一些是涉及保密信息…种种原因吧。但是又不愿意删除这些真实的自己，所以就这样吧重新打扮一个体面的自己也挺好的。

然后就是自己没打算花大力气去写，毕竟有自己的日志了。所以不写啦~各位老板有兴趣我们来私聊啊啊哈哈
